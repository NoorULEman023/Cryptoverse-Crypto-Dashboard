# CryptoVerse – Real-Time Cryptocurrency Dashboard

## Overview

CryptoVerse is a modern, responsive cryptocurrency dashboard that provides real-time market insights through an interactive and visually engaging interface. The application combines live cryptocurrency data, 3D visualizations, interactive charts, and a personalized watchlist to deliver a seamless user experience.

The project is developed using HTML5, CSS3, and Vanilla JavaScript while integrating Three.js for 3D graphics, Chart.js for data visualization, GSAP for animations, and the CoinGecko API for live cryptocurrency market data.

This project was developed as part of a **Full Stack Development Internship** to demonstrate frontend development skills, API integration, responsive web design, and modern JavaScript programming.

## Challenges Faced

During the development of CryptoVerse, several technical challenges were encountered:

- Integrating real-time cryptocurrency data from external APIs.
- Managing asynchronous API requests efficiently.
- Designing responsive layouts for multiple screen sizes.
- Implementing persistent watchlist functionality using Local Storage.
- Synchronizing live market updates without affecting user experience.
- Creating smooth animations while maintaining performance.
- Organizing JavaScript into reusable and maintainable modules.

These challenges were addressed through modular JavaScript architecture, responsive CSS techniques, and efficient DOM manipulation.
